# devportal-samples

Docker Compose configuration samples for various integration scenarios.

## Overview

| Property | Value |
| -------- | ----- |
| **Repository** | [veecode-platform/devportal-samples](https://github.com/veecode-platform/devportal-samples) |
| **Type** | Configuration only (no code) |
| **Uses Image** | veecode/devportal |

## Purpose

The `devportal-samples` repository provides ready-to-use deployment examples for:

- Authentication providers (GitHub, Azure, LDAP, Keycloak)
- SCM integrations (GitHub, Azure DevOps, Bitbucket)
- CI/CD tools (Jenkins, SonarQube)
- Kubernetes cluster configurations (local and remote)
